# BELAJAR JAVASCRIPT v2

Repositori khusus untuk meletakkan berbagai projek ku hasil dari belajar JavaScript.

Video ini adalah kursus lengkap **JavaScript** dari tingkat pemula hingga profesional oleh *SuperSimpleDev*. Kursus ini berfokus pada pendekatan berbasis proyek, di mana peserta akan membangun situs web kompleks seperti *Amazon.com* untuk mempraktikkan konsep-konsep pemrograman. 

Berikut adalah ringkasan topik utama yang dibahas:

* **Dasar-Dasar JavaScript (0:02:01):** Pengenalan tentang apa itu *JavaScript*, cara menulis kode pertama, penggunaan konsol peramban, dan pemahaman dasar tentang *syntax*.
* **Angka, Matematika, dan String (0:14:48 - 0:43:26):** Mempelajari cara mengelola angka dan teks dalam *JavaScript*, termasuk operasi matematika dan manipulasi *string*.
* **Variabel & Logika (1:51:06 - 3:34:12):** Penggunaan `let`, `const`, `var`, serta struktur kontrol seperti *booleans* dan *if-statements* untuk membuat logika program.
* **Fungsi & Objek (3:34:12 - 5:25:46):** Mempelajari cara mengelompokkan kode ke dalam fungsi dan mengelola data kompleks menggunakan objek.
* **DOM (Document Object Model) (5:25:46):** Cara berinteraksi dengan elemen *HTML* secara langsung untuk membuat situs web menjadi dinamis.
* **Array dan Loop (7:42:28 - 9:28:26):** Teknik mengolah sekumpulan data dan melakukan perulangan efisien.
* **Konsep Lanjutan (10:10:22 - 19:32:59):** Mempelajari *Advanced Functions*, modul, *Object-Oriented Programming* (OOP), *classes*, dan *inheritance*.
* **Backend & Asynchronous (19:32:59 - 22:14:08):** Pengenalan komunikasi *server* menggunakan *callbacks*, *promises*, *fetch*, serta *async/await* untuk menangani data eksternal.

Kursus ini juga mencakup persiapan lingkungan kerja (*VS Code*), pengujian (*testing*), penggunaan pustaka eksternal, dan integrasi penuh antara *HTML, CSS, dan JavaScript* untuk membangun proyek nyata.
